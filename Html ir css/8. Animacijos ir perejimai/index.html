Transform

  • scale - keicia elemento dydi (default 1)
  • ,scaleX,scaleY,scaleZ,scale(x,y), scale3d(x,y,z)
  • rotate - pasuka elementa (deg - laipsniais)
  • rotateX(x) rotateY(y) rotateZ(z), rotate(z), rotate3d(x,y,z,a)
  • translate - elemento pastumimas kazkur
  • translateX(x), translateY(y), translateZ(z), translate() translate3d(x,y,z)
  • skew - skew(x,y), skewX(x), skewY(y) (tampo kaip gumini) tempa elementa uz kampu.
  • matrix, matrix3d
  • perspective
