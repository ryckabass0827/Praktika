CSS rašymo būdai

Inline - Linijinis

Linijinis stilius yra, kai elementui, kurį stilizuojame suteikiame style atributą ir jo viduje aprašome stilius. Neturi perpanaudojimo galimybės ir yra skirtas tiktais testavimui.

Internal - Vidinis

Vidinis stilius yra, kai HTML failo viduje, head tag'o dalyje pasirašome style tag'ą ir tame style tag'e rašome CSS kodą. Turi šiek tiek perpanaudojimo galimybių (nes turi selektorius) ir dažniausiai naudojamas testavimui.

External - Išorinis

Išorinis stilius yra rašomas atskirame CSS faile ir yra prijungiamas prie HTML failo naudojant link tag'ą. Turi visas reikalingas/norimas perpanaudojimo galimybes, yra plačiausiai naudojamas ir galutinio projekto variante, turėtų likti tik šis stiliaus rašymo būdas.

Basic Selectors

Element / Tag

Selektinami visi tag'ai pagal nurodytą elementą.
Pvz.: h2 -> parinks visus h2 elementus esančius HTML'e.

Class

Klasė sukuriame kokiu tik norime logišku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, jog kuriame klasę - sugalvoto pavadinimo pradžioje parašome tašką. HTML'e norint priskirti klasę elementui, tam elementui suteikiame atributą "class" ir jam priskiriame sugalvotą klasės pavadinimą be taško.
Pvz.: CSS'e: .klasesVardas; HTML'e: <p class="klasesVardas">
Klasės turi daug su daug santykį. Vienas elementas gali turėti daugelį klasių ir viena klasė gali būti priskirta daugeliui elementų.

ID

ID sukuriame kokiu tik norime logišku ir trumpi pavadinimu. CSS'e tam, kad nurodytume, jog kuriame ID - sugalvoto pavadinimo pradžioje parašome groteles. HTML'e norint sriskirti ID elementui, tam elementui suteikiame atributą "id" ir jam prisikiriame sugalvotą ID pavadinimą be grotelių.
Pvz.: CSS'e: #idPavadinimas; HTML'e: <p id="idPavadinimas">
ID turi vienas su vienu santykį. Vienas elementas gali turėti TIK vieną ID. Ir vienas ID turėtų būti priskirtas TIK vienam elementui.

Colors

Names

Spalvos gali turėti vardus. Vardinių spalvų yra 140. Norint priskirti elementui spalvą, priskiri tos spalvos vardą.
Pvz.: red; blue; green; lightgrey; darkgreen; ...

RGB

RGB - Red Green Blue. Kiekvienos iš šių tryjų spalvų kiekio galima idėti nuo 0 iki 255 ir šitaip išgaudit bet kurią spalvą.
Pvz.: rgb(0, 0, 0) - juoda; rgb(255, 0, 0) - raudona; ...
RGBA - Red Green Blue Alpha. Alpha nurodo spalvos matomumą ir jis gali būti nuo 0 iki 1 tūkstantąją paklaida.
Pvz.: rgba(0, 255, 0, 0) - nematoma (žalia); rgba(0, 255, 0, 1) - pilnai matoma (žalia); ...

Hex

Hex - hexadecimal - šešioliktainis. Kiekvienos iš spalvos (RGB) galima nurodyti nuo 00 iki ff ir šitaip išgauti bet kurią spalvą.
Skaičiavimas šešioliktainėje sistemoje: 00 01 02 03 04 05 06 07 08 09 0a 0b 0c 0d 0e 0f 10 11 12 13 ..... 19 1a 1b ... 1f 20 21 22 ... 9f a0 a1 a2 ... a9 aa ab ac ad ae af b0 ...... ff
Pvz.: #000000 - juoda; #ff0000 - raudona; ...
Trumpiniai: #000 - juoda; #f00 - raudona; ...
Spalvos matomumas su Hex: gale prisideda dar du šešioliktainiai simboliai, kur 00 - permatomas, ff - pilnai matomas.
Pvz.: #00ff0000 - nematoma (žalia); #00ff00ff - pilnai matoma (žalia); ...
Trumpiniai: #0f00 - nematoma (žalia); #0f0f - pilnai matoma (žalia); ...

Positions

Static

Statinė pozicija nieko nedaro. Nei pakeičia elemento poziciją, nei suteikia jam galimybę naudotis top, left, right, bottom, z-index stiliais.
Statinė pozicija gali būti suteikiama kaip placeholder'is, arba panaudota tam, kad nuimti kokią kitą poziciją.

Absolute

Elementas su absolute pozicija išeina iš puslapio tėkmės. Suteikia galimybę naudotis top, left, right, bottom, z-index stiliais.
Top, left, right arba bottom nustatymas pastumia elementą per nurodytą atstumą, nuo nurodytas kraštinės, nuo pradinio viewport (ekrano vaizdo) ARBA nuo artimiausio protėvinio elemento, kuris turi relatyviąją poziciją.
