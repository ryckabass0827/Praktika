Tėvinis elementas gali turėti daugelį vaikinių elementų.
Vaikinis elementas, gali turėti TIK vieną tėvinį elementą.

Paveldėjimas

Paveldėti stilius, gali tik vaikinis elementas iš tėvinio elemento.
Paveldimi stiliai yra tie, kurie keičia elemento išvaizdą. Pvz.: color, font-size, font-family...
Nepaveldimi stiliai yra tie, kurie keičia elemento būseną. Pvz.: width, height, display, border, position...
Net paveldėti stiliai, gali nebūti atvaizduoti dėl stilių prioritetų.

Numatytieji stiliai

Numatytieji stiliai yra tag'ams automatiškai naršyklės suteikti stiliai. Tarp skirtingų naršyklių, numatytieji stiliai gali minimalistiškai keistis ir šitaip sugadinti svetainės vaizdą skirtingų naršyklių vartotojams.
Tam, kad nebūtų problemų kuriant dizainą skirtingoms naršyklėms, yra naudojami Reset arba Normalize CSS'ai.
Reset - resetina visus stilius (ištrina stilius).
Normalize - suvienodina visus stilius tarp naršyklių ir panaikina dalį kenkėjiškų stilių.

Esant dviem selektoriam, kurie nusako to paties elemento vienodus stilius, yra atsižvelgiama į tų selektorių prioritetus. Aukštesnį balą turintys prioritetai, turi aukštesnį prioritetą.
Esant dviem vienodo prioritetų balo selektoriams, keičiant tą patį elemento stilių, yra atsižvelgiama į CSS eiliškumą. Kuris CSS stilius aprašytas vėliau, tas ir bus pritaikomas.

Jeigu įmanoma - nenaudoti!!! !important prierašas prie stiliaus ignoruoja visus prioritetus ir suteikia tą stilių ir niekas jo negali pakeisti, apart kito !important prierašo, kuris turi aukštesnį prioritetą.
