Responsive Design

Responsive Design gali reaguoti į vartotojo įrenginį ir/arba jo plotį.

Esant skirtingiems ekranų dydžiams - galima pritaikyti skirtingus stilius naudojant @media query.

Breakpoint devices

  1. portrait phone
  2. table / landscape phone
  3. laptop
  4. desktop

Breakpoint width (px)

  1. <500px
  2. <678px
  3. <992px
  4. <1200px
  5. <1400px
  6. >1400px

Mobile First

  • is mazesnio ekrano dydzio yra lengvesnis perejimas i didesni (lengviau prideti naujo turinio negu atimti sena)
  • koncentracija i turini (del mazo vietos kiekio)
  • didesnis kiekis vartotoju naudoja mobiliasias aplikacijas (orientuojames i didesni vartotoju kieki)
  • Google paieskos algoritmas geriau ivertina mobile first dizaino kurtas svetaines

Naudojant Mobile first dizaino principa pradedame nuo maziausio ekrano ir judame link didesniu.
tam naudojame @media (min-width:____px).
