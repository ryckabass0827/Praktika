Pseudo selektoriai

  • *
  • element
  • .class
  • #id
  • selector1, selector2
  • selector1 selector2
  • selector1 > selector2
  • element.class#id - elementas turintis nurodytą klasę/es ir/arba nurodytą id
  • selector1 + selector2 - iš karto už selector1 einantis selektor2 (tame pačiame lygyje)
  • selector1 ~ selector2 - už selector1 visi einantys selector2 (tame pačiame lygyje)
  • [attribute] - atributas
  • [attribute=value] - atributas turintis nurodytą reikšmę
  • [attribute~=value] - atributas, kurio reikšmėje yra nurodytas žodis
  • [attribute*=value] - atributas, kurio reikšmėje yra nurodyta žodio dalis
  • [attribute|=value] - atributas, kurio reikšmė yra value arba value-kazkas
  • [attribute^=value] - atributas, kurio reikšmė prasideda value...
  • [attribute$=value] - atributas, kurio reikšmė pasibaigia ...value
  • :link - paprasta/nelankyta nuoroda
  • :visited - lankyta nuoroda
  • :hover - užėjęs kursorius
  • :active - aktyvi (paspaudimo momento) nuoroda
  • element:empty - tuščias elementas (be content)
  • :not(selector) - neparenka nurodyto selektoriaus
  • :first-child - parenka pirmąjį vaikinį elementą
  • :last-child - parenka paskutinįjį vaikinį elementą
  • :only-child - parenka vienintelį vaikinį elementą (jeigu jis yra vienas)
  • :first-of-type - parenka pirmąjį nurodyto tipo elementą
  • :last-of-type - parenka paskutinįjį nurodyto tipo elementą
  • :only-of-type - parenka vienintelį nurodyto tipo elementą
  • :nth-child(n) - parenka nurodytą/us vaikinį/ius elementą/us
  • :nth-last-child(n) - nuo galo parenka nurodytą/us vaikinį/ius elementą/us
  • :nth-of-type(n) - parenka nurodytą/us elementą/us pagal tipą
  • :nth-last-of-type(n) - nuo galo parenka nurodytą/us elementą/us pagal tipą
  • ::marker - sąrašo elementų žymės
  • ::before - sukuria kažkokį turinį prieš elementą
  • ::after - sukuria kažkokį turinį už elemento
  • input:checked - pazymetus/parinktus radio/checkbox/option pasirinkimus
  • input:default - is anksto parinktas pasirinkimas
  • input:enabled/disabled - inputas kuris turi/neturi enable/disable atributo
  • input:focus - inputas ant kurio yra paspausta arba kuris yra fokusuotas
  • input:in-range - inputas kurio verte kurio verte yra nurodytose ribose
  • input:out-of-range - inputas kurio verte yra uz nurodytu
  • input:indeterminate - inputas kurio reiksme neapibresta
  • input:invalid - inputas kurio verte yra netinkama
  • input:valid - inputas kurio verte yra tinkama
  • input:reqiured - inputas su requared atribudu
  • input:optional - inputas be requared atributo
  • input:read-only - inputas su read-only atributu
  • input:read-write - inputas be read only atributo
  • input::placeholder - ivesties laukelio placeholder atributo reiksmes parinkimas stilizavimui
  • selector::first-letter - parenka elemento pirmaja raide
  • selector::first-line - parenka elemento pirmaja eilute
  • ::selection - parenka vartotojo pazymeta turini
  • :is()
  • :has()
  • :where()
